# KernelSU-Redmi12c
KernelSU for Redmi12c unofficial
Это неофициальный порт KernelSU для Redmi 12c aether earth 
Я как автор не ручаюсь за телефоны которые могут окирпичиться после этой процедуры
Проверялось на Redmi12c aether Lmodroid 15

Инструкция
Скачиваете файл KernelSU-Redmi12c.zip прошиваете в рекавери дальше заходите в систему и ставите kernelSU-next и наслаждаетесь рут правами
